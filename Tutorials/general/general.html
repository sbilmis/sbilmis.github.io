1. Combining PDF Files

Combining multiple PDF files can be achieved using different command-line tools. Below are two effective methods:

1.1. Using `pdfunite`

A simple and straightforward way to merge PDF files: 

pdfunite path/to/fileA.pdf path/to/fileB.pdf path/to/merged_output.pdf

1.2. Using `gs` (Ghostscript)

A more powerful method, especially for advanced needs like optimizing the output: 

gs -q -dNOPAUSE -dBATCH -sDEVICE=pdfwrite -sOutputFile=merged.pdf source1.pdf source2.pdf source3.pdf
  • For more details, see: StackExchange: Combining PDFs Using Command Line

2. Converting ENEX Files to Org Mode

Export Evernote `.enex` files to Org mode using the `orgenex` tool:

  • Tool: orgenex

  • Install via pip: 

    pip install orgenex

  • Convert an `.enex` file to Org mode: 

    orgenex input.enex output.org
