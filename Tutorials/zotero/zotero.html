1. What is Zotero?

  • Zotero is a free, open-source tool for managing references, citations, and research materials. It allows users to collect, organize, cite, and share research using its desktop application, browser connector, and cloud sync.

2. WebDAV Sync

  • Zotero supports syncing libraries via WebDAV, which allows users to store their attachments on a remote server instead of Zoteroâ€™s cloud storage.
  • pcloud and koofr have webdav service.
  • Be cautious when using WebDAV with multiple profiles, as it may cause sync issues if different folders are used.

3. Using Multiple Profiles

  • Zotero allows the creation of multiple profiles for managing different libraries or accounts. Use the `-P` option when launching Zotero to choose a profile: 

        zotero -P
  • Example: The `sbilmismybooks` profile was created using the account `sbilmis@outlook.com`.
  • Caution: Using multiple profiles with WebDAV sync may lead to issues if WebDAV storage does not support separate folders for different profiles.

4. Addons for Zotero

  • Zotero Addons Manager:
    • Install and manage Zotero addons easily, including version-specific addons for different Zotero versions.
  • Zotero Better Notes:
    • Organize and annotate notes directly within Zotero to improve research note management.
  • ZotFile:
    • Enhance file management by renaming, moving, and syncing PDFs with tablets.
  • Zotero Folder Import:
    • Import a whole folder of files into Zotero.
  • Zotero Storage Scanner:
    • Scan Zotero storage for missing or unlinked files.
  • Zotxt:
    • Integrate Zotero with Emacs for inserting citations into documents.
  • Zutilo:
    • Add functionalities like copying file paths, batch editing, and shortcut enhancements.
  • Zotero Inspire:
    • Integrate Zotero with INSPIRE-HEP for physics-related literature.
  • Zotero Preview:
    • Add a preview pane to view bibliography entries directly within Zotero.
  • Zotero OCR:
    • Add OCR capabilities to Zotero for extracting text from scanned PDFs.

5. Additional Notes

  • Use addons wisely to enhance productivity and tailor Zotero to specific needs.
  • Official Zotero Plugin Directory provides a comprehensive list of supported and tested plugins.
