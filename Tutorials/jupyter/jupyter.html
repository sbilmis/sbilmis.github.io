1. Listing Installed Jupyter Kernels

  • To list all installed Jupyter kernels: 

        jupyter kernelspec list
  • When using Org-Babel for IPython (Emacs-Jupyter), specify the kernel in the `:kernel` option. For example, use `:kernel python3` if your kernel name includes "ipython".
  • Example Output: ``` Available kernels: python3 /opt/homebrew/lib/python3.11/site-packages/ipykernel/resources matlab38 /Users/sbilmis/Library/Jupyter/kernels/matlab38 test2 /Users/sbilmis/Library/Jupyter/kernels/test2 wolframlanguage13 /Users/sbilmis/Library/Jupyter/kernels/wolframlanguage13 sagemath-10.0 /usr/local/share/jupyter/kernels/SageMath-10.0 ```

2. Listing Running Jupyter Notebooks and Kernels

  • To list running Jupyter notebooks and kernels:

    • See StackOverflow discussion for more details
        jupyter notebook list

3. Giving Presentations with Jupyter Notebooks

  • Use the RISE extension to turn a Jupyter notebook into a live, `reveal.js`-based presentation:
    • RISE Documentation
  • RISE allows for interactive presentations directly from the notebook, enabling smooth transitions, full-screen viewing, and slide navigation within the notebook environment.
