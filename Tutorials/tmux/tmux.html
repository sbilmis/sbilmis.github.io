1. Tmux Addons

  • Popular Addons for enhancing tmux functionality:
    • `tmuxinator`: Manage complex tmux session configurations.
    • `tmux-continuum`: Automatically save tmux sessions at regular intervals.
    • `tmux-resurrect`: Restore tmux sessions exactly as they were after closing.

2. Installing Tmux and Addons

  • Installing tmux:

    • On macOS: 

            brew install tmux

    • On Ubuntu: 

            sudo apt update
      sudo apt install tmux

2.1. Installing Addons

  • Tmuxinator:
    • `tmuxinator` simplifies managing tmux sessions and layouts.

    • Installation via `gem`: 

            # First, install Ruby if not installed
      sudo apt install ruby  # Ubuntu
      # Install tmuxinator
      gem install tmuxinator
      # After installation, run tmuxinator to confirm it's installed
      tmuxinator
  • Tmux Continuum:
    • `tmux-continuum` automatically saves tmux sessions at regular intervals.
    • Requires `tmux-resurrect` to function.
    • Installation:

      1. Clone the repository: 

                 git clone https://github.com/tmux-plugins/tmux-continuum ~/.tmux/plugins/tmux-continuum

      2. Add the following lines to your `.tmux.conf` to enable automatic session saving: 

                 set -g @continuum-restore 'on'
         set -g @continuum-save-interval '15'  # Saves every 15 minutes

      3. Reload your tmux configuration: 

                 tmux source-file ~/.tmux.conf
  • Tmux Resurrect:
    • `tmux-resurrect` restores tmux sessions exactly as they were, preserving window layouts and processes.
    • Installation:

      1. Clone the repository: 

                 git clone https://github.com/tmux-plugins/tmux-resurrect ~/.tmux/plugins/tmux-resurrect

      2. Add the following to your `.tmux.conf` to configure resurrect: 

                 # To save sessions
         bind r source-file ~/.tmux/plugins/tmux-resurrect/resurrect.tmux
         set -g @resurrect-capture-pane-contents 'on'

      3. Reload your tmux configuration to activate: 

                 tmux source-file ~/.tmux.conf
      4. To save and restore sessions:
        • Press `Prefix + Ctrl-s` to save the current tmux session.
        • Press `Prefix + Ctrl-r` to restore the last saved tmux session.

2.2. Additional Notes

  • Configuration Reload:

    • After making changes to `.tmux.conf`, reload tmux settings with: 

            tmux source-file ~/.tmux.conf
  • Help and Documentation:
    • Use `man tmux` to access tmux manual pages for detailed command information.
    • Addon documentation:
      • tmuxinator
      • tmux-continuum
      • tmux-resurrect
