1. What is DEVONthink?

DEVONthink is a knowledge management tool designed for storing, organizing, and retrieving documents efficiently. It supports various file types, such as PDFs, web archives, text documents, and images, making it an essential tool for researchers, writers, and professionals.

1.1. Key Uses of DEVONthink

  • Document management and organization.
  • Advanced search capabilities.
  • Importing, indexing, and tagging documents for easy retrieval.

1.2. Advantages of DEVONthink

  • Flexible: Supports both imported and indexed documents.
  • Powerful search: Offers precise and fast search capabilities.
  • Automation: Automates organization with smart rules and scripts.
  • Sync: Keeps data synchronized across devices.

1.3. General Disadvantages

  • Steep learning curve for new users.
  • Requires periodic organization and database management.
  • Sync setup can be complex initially.

2. Importing vs. Indexing in DEVONthink

Read Help > In & Out section before you decide your workflow.

DEVONthink allows two main ways to manage files:

Method Description Advantages Disadvantages
Import Copies files into DEVONthink. - Local access. - Consumes storage space.
    - Files are always in the database. - Risk of duplicates if unmanaged.
Index Files remain in their original location. - Saves storage space. - Relies on external file paths.
  DEVONthink only links to the files. - Updates reflect automatically. - Files might break if moved.

2.1. Points to Be Careful About:

  • Imported files: Manage duplicates to avoid clutter.
  • Indexed files: Avoid changing the folder structure outside DEVONthink; this can break the links.
  • Files synced by other applications: If you are indexing files in a folder synced by another application, e.g., iCloud or Dropbox, you may need to use the File > Update Indexed Items command to manually update the indexed group. DEVONthink does this to avoid causing a discrepancy in the data for the other application. Also, a particular process may not generate a filesystem event for DEVONthink to detect a change has occurred.
  • Backup your database regularly to prevent data loss.

2.2. When to Use Importing vs. Indexing

  • If you are using files primarily outside DEVONthink, indexing is the way to go. It allows you to manage and access your files without duplication while keeping the folder structure intact.

3. How to Tell If a Database is Indexed or Imported

You can quickly determine if a database uses imported or indexed files:

  1. Navigate to Tools > Filter > Info pane.
  2. Use the File Location dropdown menu for interactive filtering of the current location.

This provides a clear indication of whether files are stored (imported) in the database or linked (indexed) to external locations.

For more details, refer to the DEVONthink official help: DEVONthink Help Page.
