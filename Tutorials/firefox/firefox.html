1. Firefox Tips and Tricks

1.1. Keyboard Shortcuts

Efficiently navigate and manage your tabs using these keyboard shortcuts:

  • `Ctrl-Tab`: Switch to the previous tab (recently used tab).
  • `Ctrl-Shift-Tab`: Open a list of all open tabs (similar to `Cmd-Shift-A` in Edge).

1.2. Zooming with CMD (Ctrl) + Mouse Wheel

Enable zooming functionality using the `CMD` (or `Ctrl`) key and the mouse wheel:

  1. Open the advanced configuration settings in Firefox by typing `about:config` in the address bar.

  2. Set the following option: 

       # Locate and change the value of this setting:
   mousewheel.with_meta.action to 3
   # (Default value is 1)
