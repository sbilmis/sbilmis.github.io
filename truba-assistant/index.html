Install software without sudo

I am using a shared HPC cluster running Rocky Linux 9.4. I do not have sudo/root privileges. SLURM is used as the job scheduler and Lustre is the filesystem. I want to install [software name and version] in my own directory, preferably under $HOME or a project/work directory. Please give me a safe user-space installation plan. Do not suggest sudo, yum, dnf, apt, or system-wide changes. Include commands and explain how to test the installation.

Write a SLURM job script

I am using a Rocky Linux 9.4 HPC cluster with SLURM and Lustre. I want to run [program or command] using [number of CPU cores], [memory amount], and [time limit]. I do not have root privileges. Please write a minimal sbatch script, explain each #SBATCH line, and show how to submit the job and check its output.

Create a Python environment

I am using a Rocky Linux 9.4 HPC cluster with SLURM and Lustre. I do not have sudo/root privileges. I need a Python environment for [packages or project]. Please suggest a user-space setup using conda/mamba, venv, or pip without sudo. Include commands, how to activate the environment inside an sbatch script, and how to avoid filling my home quota.
